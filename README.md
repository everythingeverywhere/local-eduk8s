# local-eduk8s
To stand up eduk8s locally on minikube


Clone GitHub repo:

```copy
git clone
```

Deploy the minkube cluster and [eduk8s](https://docs.eduk8s.io/):

```copy
cd ./local-eduks8 && ./deploy.sh
```

If desired you can check eduk8s with:

```
/.deploy_check.sh
```

Delete the cluster and eduk8s

```copy
./delete.sh
```
